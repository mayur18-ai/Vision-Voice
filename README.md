# 👁️ VisionVoice AI

An AI-powered web application designed to assist users (especially visually impaired individuals) with voice-based navigation and real-time interaction.

---

## 🚀 Live Demo
🔗 https://vision-voice-five.vercel.app

---

## 📌 Features

- 🎤 Voice Commands (Start / Stop Navigation)
- 🔊 Audio Feedback using Speech Synthesis
- 📷 Camera Access for real-time interaction
- 🔍 Scan functionality for environment awareness
- 🚨 Emergency button for quick alerts
- 🔄 Flip Camera support
- 🌐 Multi-language support (English / Hindi)

---

## 🧠 How It Works

1. User gives voice command  
2. Application processes input using browser APIs  
3. Camera and system respond accordingly  
4. Audio feedback is provided to the user  

---
